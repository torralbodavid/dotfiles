# 🚀 Dotfiles

Personal macOS configuration setup. Clone, run, and your Mac is ready to work.

## ⚡ Quick Start

```bash
git clone https://github.com/tu-usuario/dotfiles.git ~/Code/dotfiles
cd ~/Code/dotfiles
chmod +x install.sh
./install.sh
```

That's it. The script handles **everything**.

## 📦 What does `install.sh` do?

The installer runs these steps automatically and interactively:

### 1. 🍺 Installs Homebrew
If you don't have Homebrew installed, it installs it automatically (compatible with Apple Silicon and Intel).

### 3. 🔧 Configures Git
It asks for your **name** and **email** to generate `~/.gitconfig` with the proper settings. It also links `.gitignore_global`.

### 4. 🔗 Creates Symlinks
It creates symbolic links from your `$HOME` folder to the configuration files in this repository:

```
~/.zshrc             → dotfiles/configs/.zshrc
~/.gitconfig         → dotfiles/configs/.gitconfig
~/.gitignore_global  → dotfiles/configs/.gitignore_global
```

> If these files already exist, an automatic backup is created (`.backup`).

### 5. 🔒 Local Secrets (`.zsh_aliases`)
It will ask if you want to set up private SSH aliases (e.g., `par01`, `mia03`), which are then saved to `~/.zsh_aliases`. This file is loaded dynamically by `~/.zshrc` but is **never tracked by Git** to keep your secrets safe.

### 6. 🍎 Applies macOS Preferences
It runs `macos/defaults.sh` which sets up:

- **Finder:** Shows hidden files and file extensions
- **Trackpad:** Tap-to-click enabled
- **Dock:** Size 48, positioned at the bottom, auto-hide enabled
- **Keyboard:** Fast key repeat rate
- **Screenshots:** Saved inside `~/Screenshots`

## 📂 Project Structure

```
dotfiles/
├── Brewfile                    # Homebrew apps (base)
├── install.sh                  # Interactive installation script
├── README.md                   # This file
├── configs/
│   ├── .zshrc                  # Shell aliases and functions
│   ├── .gitconfig              # (generated by install.sh)
│   └── .gitignore_global       # Global Gitignore
└── macos/
    └── defaults.sh             # macOS system preferences
```

## 🔄 Updating

If you change the configuration files in this repository, the changes apply instantly thanks to the symlinks. You just need to:

```bash
source ~/.zshrc   # To reload aliases
```

To re-apply macOS preferences:
```bash
bash ~/Code/dotfiles/macos/defaults.sh
```

## 📝 Notes

- The `.gitconfig` **is not pushed to the repository** because it contains personal data (name, email, `$HOME` path). It's generated during installation.
- Any private functions or aliases should be placed in `~/.zsh_aliases`, which keeps them out of version control.
- Compatible with **macOS Monterey** or newer.
- Compatible with **Apple Silicon (M1/M2/M3/M4)** and **Intel**.
